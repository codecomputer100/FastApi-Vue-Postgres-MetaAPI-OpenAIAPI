
# Meta Ads AI Dashboard

Fullstack project to manage and optimize **Meta Ads (Facebook / Instagram)** campaigns using AI.

* **Backend:** FastAPI + SQLAlchemy + PostgreSQL
* **Frontend:** Vue 3 + PrimeVue + Sakai template
* **External APIs:** Meta Marketing API + OpenAI API

---

## What does this project do?

* Lists the Meta Ads campaigns of the connected user.
* Shows performance metrics per campaign (impressions, clicks, CTR, CPC, CPM, etc.).
* Generates an AI-based (OpenAI) campaign configuration recommendation in JSON format, including:

  * Suggested campaign name
  * Recommended objective (Sales, Leads, Traffic, etc.)
  * Estimated budget
  * Targeting (location, age, interests, etc.)
  * Suggested ad format and copy
